# OS_Colloquium_1

Проект содержит реализацию трех алгоритмических задач на языке C++

## Технологический стек

*   **Язык:** C++17
*   **Система сборки:** CMake

## Описание задач

### 1. Генерация последовательности факториалов

Разработать функцию, которая принимает на вход натуральное число `n` и возвращает контейнер, содержащий первые `n` факториалов.

### 2. Удаление дубликатов с сохранением порядка

Разработать функцию, которая принимает на вход контейнер (массив) с числами и возвращает новый контейнер без дубликатов.

### 3. Рекурсивный реверс связного списка`

Разработать структуру односвязного списка и реализовать функцию разворота (инверсии) списка, используя рекурсию.

## Сборка и запуск

Для сборки проекта требуется CMake версии 3.24 или выше и компилятор с поддержкой C++17.

**Команды для сборки:**

```bash
mkdir build
cmake -B build
cmake --build build
